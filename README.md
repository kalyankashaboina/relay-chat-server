# Relay

Relay is a scalable real-time chat backend built with Node.js, TypeScript, Express, Socket.IO, and MongoDB.

## Features
- Real-time messaging with Socket.IO
- JWT-based authentication
- MongoDB as the source of truth
- Modular, scalable architecture
- Redis-ready design for horizontal scaling

## Tech Stack
- Node.js + TypeScript
- Express
- Socket.IO
- MongoDB
- Redis (planned)

## Getting Started

```bash
npm install
npm run dev
